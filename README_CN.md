## 什么是一人公司？

一人公司是指：

- 由单人即可构建与运营
- 有可持续收入
- 依赖软件、内容或杠杆，而非雇佣劳动力
- 追求自主与自由，而非盲目规模化

本清单关注**独立创业者与可持续盈利的小生意**，而非风投驱动的高速扩张公司。
